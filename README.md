# Amazon Prime Video Analysis

##  Project Overview

This project presents an interactive Power BI dashboard analyzing the
Amazon Prime Video dataset. The objective is to explore content
distribution, ratings, genres, country availability, and release trends
to generate meaningful business insights.

##  Key Objectives

-   Analyze the distribution of Movies and TV Shows.
-   Examine content ratings (13+, 16+, 18+, PG-13, etc.).
-   Identify most common genres on the platform.
-   Explore country-wise content availability.
-   Study release trends over time.
-   Build interactive dashboard for dynamic filtering.

------------------------------------------------------------------------

##  Dashboard Insights

###  Content Type Distribution

-   Movies account for approximately 80% of the platform's content.
-   TV Shows contribute around 20% of total titles.

###  Ratings Analysis

-   Majority of content falls under 13+ and 16+ categories.
-   Lower contribution from PG-13 and R-rated content.

###  Genre Analysis

-   Drama and Comedy are among the most frequently listed genres.
-   Diverse genre distribution supports broad audience targeting.

###  Country-Wise Distribution

-   Content is widely distributed across multiple countries.
-   Strong presence in North America, Europe, and Asia.

###  Release Year Trend

-   Significant increase in content production after 2000.
-   Rapid growth observed after 2010, indicating platform expansion.

##  How to Run This Project

1.  Download the `.pbix` file from this repository.
2.  Install Power BI Desktop.
3.  Open the file using Power BI Desktop.
4.  Refresh data if required.
5.  Use filters to interact with the dashboard.
